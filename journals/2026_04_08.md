# Optimization
collapsed:: true
	- Optimization Problem:
	- $$\text{minize } f(\vec x)\\\text{subject to } \vec x \in \mathcal{S}$$
	- The vector $\vec x \in \mathbb{R}^n$ is the optimization variable
	- The set $\mathcal{S} \subseteq \mathbb{R}^n$ is the constraint or feasible set
	- The function $f:\mathbb{R}^n \rightarrow \mathbb{R}$ is the objective function or **cost function**. This function is **not given to us**.
- > Discrete or Continuous optimization is decided by whether the constraint set is discrete or Continuous
- # Discrete Optimization
	- $\mathcal{S} = \set{x_1, x_2, ...}$ can be finite ($|\mathcal{S}| < \infty$) and/or countable ($|\mathcal{S}| = |\mathbb{N}|$)
	- There are 2 formulations of this problem
		- Binary optimization problem: $\mathcal{S} \subseteq \set{0,1}^n$
		  logseq.order-list-type:: number
		- Integer optimization problem: $\mathcal{S} \subseteq \mathbb{Z}^n$
		  logseq.order-list-type:: number
	- Examples of such optimization problems:
		- *Shortest Path Problem* - Finding the shortest problem from a source vertex $$s \in \mathcal{V}$$ to a destination vertex $$t \in \mathcal{V}$$ in a directed graph $\mathcal{G} = (\mathcal{V,E})$ where the cost of a path is the sum of weights of all the constituent paths. This is formulated as a binary optimization problem where the binary variable $$x_e$$ is whether or not a particular edge $$e \in \mathcal{E}$$ is selected.
		  logseq.order-list-type:: number
			- Binary Variable $x_e \in \set{0,1}^n$ where $n= |\mathcal{E}|$ is the number of edges
			  logseq.order-list-type:: number
			- Constraints
			  logseq.order-list-type:: number
				- Out flow of source vertex $\mathcal{O}_s$ should be non-zero $\sum\limits_{e\in \mathcal{O}_s}x_e = 1$
				  logseq.order-list-type:: number
				- In flow of destination vertex $\mathcal{I}_d$ should be non-zero $\sum\limits_{e\in \mathcal{I}_d}x_e = 1$
				  logseq.order-list-type:: number
				- For every other vertex $v \in \mathcal{V} \setminus \set{s,t}$, $\sum\limits_{e\in \mathcal{O}_v}x_e = \sum\limits_{e\in \mathcal{I}_v}x_e$
				  logseq.order-list-type:: number
			- Objective function is $$f(x) = \sum\limits_{e\in\mathcal{E}}w_e x_e$$ which we minimize
			  logseq.order-list-type:: number
